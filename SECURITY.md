# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | ✅ Active support  |

## Reporting a Vulnerability

If you discover a security vulnerability in ImageUpscaler, please report it responsibly:

1. **Do NOT** open a public GitHub issue for security vulnerabilities
2. Email the maintainer directly or use GitHub's private vulnerability reporting
3. Include a detailed description of the vulnerability
4. Include steps to reproduce if possible

## Response Timeline

- **Acknowledgment**: Within 48 hours
- **Initial assessment**: Within 1 week
- **Fix or mitigation**: Within 2 weeks (depending on severity)

## Safe Harbor

We consider security research conducted in accordance with this policy as:

- Authorized and will not pursue legal action
- Exempt from DMCA restrictions
- Conducted in good faith

## Scope

This policy covers:
- The ImageUpscaler repository and its components
- The Colab notebook and its execution environment
- The `src/` package modules

Thank you for helping keep ImageUpscaler secure!
